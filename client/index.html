Assignment Details

Nitish Nashine,

Overview: You will build a self-contained document scanning and matching system with a built-in credit system. Each user has a daily limit of 20 free scans, and additional scans require requesting more credits. The project must be completed within 10 days. Using the internet for research is allowed, but copying code from online sources or peers will result in disqualification. Bonus: Implement AI-powered document matching using OpenAI, Gemini, or DeepSeek for better results.

Assignment Timeline (10 Days):

  • Day 1-2: Set up the project (Backend, Frontend, Database)
  • Day 3-4: Implement User Authentication & Role Management
  • Day 5-6: Develop the Credit System (Daily Free Credits & Admin Approval)
  • Day 7-8: Implement Document Upload & Text Matching (Basic Version)
  • Day 9: Build Smart Analytics Dashboard & Optimize Features
  • Day 10: Testing, Documentation, and Final Submission

Assignment Requirements:

  • User Management & Authentication
  • Credit System
  • Document Scanning & Matching
  • Smart Analytics Dashboard
  • API Endpoints

Tech Stack:

  • Frontend: HTML, CSS, JavaScript (No frameworks)
  • Backend: Node.js (Express) without external libraries
  • Database: SQLite (or JSON files for small-scale storage)
  • File Storage: Store documents locally
  • Authentication: Basic username-password login (hashed passwords)
  • Text Matching Logic: Used Naturals to identify similar text
